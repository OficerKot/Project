Объединяет в себе все половинки домино, которые могут появиться в игре. Осуществляет поиск по id, а также по рисунку и номеру.
Поля:
public List<DominoData> available - доступные для генерации половинки домино, которые игрок получил в процессе игры
public List<DominoData> basic - все базовые половинки домино, которые могут быть сгенерированы с самого начала игры
public Dictionary<ImageEnumerator, int> order - приоритеты рисунков (могут использоваться для сортировок, например в скрипте SigilsMenu для разделения рисунков по группам)

Методы:
DominoData GetDominoByID(string id) - возвращает информацию о домино с указанным id
DominoData GetRandomDomino() - возвращает информацию о любом домино из доступных игроку
DominoData GetDomino(ImageEnumerator image, int number) - возвращает информацию о домино с указанным рисунком и номером
bool HasAvailable() - доступны ли на текущий момент домино помимо базовых
