На сцене есть объект Inventory (всегда в единственном экземпляре), на который был добавлен данный скрипт. Он хранит в себе список (если точнее, то словарь) предметов, находящихся в данный момент у игрока. Хранение предметов в виде словаря было выбрано из-за удобства поиска по ID, а также для реализации механики крафта, так как у этой структуры доступна проверка на подмножества.
Этот скрипт взаимодействует с UIInventory для визуального отображения текущего состояния

Поля: 
public const int MAX_SIZE - количество ячеек в инвентаре
Dictionary<string, int> itemsID - предметы в инвентаре
public static Inventory Instance

Методы:

public:
AddItem() - добавление предмета в инвентарь
RemoveItem() - удаление предмета из инвентаря
Dictionary<string, int> GetCurItems() - возвращает все ID предметов в инвентаре
isFull() - проверка на заполненность
